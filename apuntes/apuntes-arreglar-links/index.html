Recopilación de etiquetas

CSS

Crear nota en CSS:
/* texto */

width: definir anchura

height: definir altura

max-width: adaptar texto hasta un % de la pantalla.
Ejemplo: max-width:80%

margin: dar margen exterior. Podemos escribir las direcciones en el sentido de las agujas del reloj (arriba, derecha, abajo e izquierda).
Si solo escribimos un número se aplicará a los 4 lados, si queremos que solo se aplique arriba y abajo, por ejemplo, se puede escribir: margin: 40px, 0. Ya que el 0 hará que se anule en ese campo.

padding: igual que margin pero para interior del cajón.

font-size: tamaño fuente

font-weight: para definir grosor del texto. 700 para negrita y 900 para super negrita. 500 para texto normal.

font-family: cambiar tipologia de fuente.

letter-spacing: útil para dejar espacio entre las letras, poner pocos px, como 1 o 2

text-align: center: centrar texto, solo vale para texto

box-sizing: border-box; PREGUNTAR A SATORY

display: puede ser inline, block, inline-block o el maravilloso flex

display flex:
justify-content: center: centrar horizontal
align-items: center: centrar vertical
flex-direction: column o row: cambiar sentido de los componentes

explicación flex de Satory:
1º - Elegir el eje principal
flex-direction: row;
2º - Justify-content - Alinea y/o distribuye los elemenetos en el eje principal
justify-content: space-between;
3º - Align-items - Alinea y/o distribuye los elemenetos en el eje secundario
align-items: center;
4º - Flex-wrap - Especifica si los elementos saltan de línea o no
flex-wrap: wrap;

variables:
Para hacer variables se pone --etiqueta: color/tipografia, de esta manera nos aseguramos de que más adelante cuando usemos las variables en nuestro CSS (ej: background-color: var(--blanco)) se podrán cambiar facilmente todas sin tener que ir una a una.
ejemplo variables:
--blanco: #fff;
--negro: #000;
--roboto: "Roboto", sans-serif;

list-style: none: quitar puntitos a las listas, si esto lo combinas con display: inline puedes hacer menús de navegación bien guapardos

botón: ejemplo de crear botón:
margin: 40px 0;
width: 200px;
height: 44px;
padding-top: 4px;
font-size: 13px;
letter-spacing: 3px;
font-weight: 700;
text-transform: uppercase;
border: 0px; (para quitar el bordecillo de botón de Windows XP)
border-radius: 22px;
